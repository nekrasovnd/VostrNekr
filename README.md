# Web Calculator

Минималистичный веб-калькулятор с базовыми функциями.

## Возможности
- сложение  
- вычитание  
- умножение  
- деление  
- очистка  
- удаление символа  

## Структура проекта
- `src/` — исходный код (HTML/CSS/JS)
- `docs/` — документация
- `tests/` — тесты
- `.github/workflows/` — CI/CD
